MUST TRY PYTHON PACKAGES

Here are some MUST TRY python packages for everyone:

  1. Pillow : For batch operation on images
  2. Dash : For building data visualisations
  3. JmesPath : Allows you to declaratively specify how to extract elements from a JSON document
  4. Requests : Makes web requests really simple
